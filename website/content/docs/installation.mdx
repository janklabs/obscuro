## Install Script

The fastest way — just run:

```sh
curl -sSL https://raw.githubusercontent.com/janklabs/obscuro/main/install.sh | bash
```

This grabs the latest release binary for your platform from GitHub.

## Build from Source

You'll need Go 1.21 or later.

```sh
git clone https://github.com/janklabs/obscuro.git
cd obscuro
go build -ldflags "-X github.com/janklabs/obscuro/internal/version.Version=$(git describe --tags --always)" -o obscuro .
```

Then move it somewhere on your PATH:

```sh
sudo mv obscuro /usr/local/bin/
```

The `-ldflags` part is important — without it, `obscuro version` won't know what version it is.

## Verify

Make sure everything worked:

```sh
obscuro version
```
